# e-com-sales-analysis
EDA project analyzing UK e-commerce retail data (2010-2011). Customer segmentation using RFM analysis, revenue trends, product performance. Python, Pandas, Matplotlib, Seaborn
# This project contains multiple insights, detailed explanations, visualizations, and business decisions. I’ve documented everything clearly inside the notebook itself, so please go through the project for a complete understanding of the findings and recommendations

## E-Commerce Sales Analysis
### A Data Analyst Case Study , UK Online Retail (2010-2011)
**By:** Hardeep Singh Dhaliwal  
**Tools:** Python - Pandas - Matplotlib - Seaborn  
**Dataset:** https://www.kaggle.com/datasets/carrie1/ecommerce-data


## Questions Answered

1. Which countries are driving the majority of orders?
2. How does revenue trend across months. is there seasonality?
3. Which products generate the most revenue?
4. How many customers are repeat buyers vs one-time buyers?
5. Who are the most valuable customers? (RFM Segmentation)

## Cleaning steps
- Removed 135,080 rows with missing CustomerID (guest checkouts)
- Removed negative Quantity rows (returns/cancellations)
- Created Revenue column: Quantity × UnitPrice

---

## Key Findings

- UK Dominates Orders
- Strong Seasonal Pattern
- Top Products
- Strong Customer Loyalty
